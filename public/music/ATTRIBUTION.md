# Music attribution

Background-music tracks bundled with the PokerTH web client. Every track is
royalty-free under the licence noted below; the same credit is also shown
in-app inside the Music panel. The track list is data-driven (`tracks.json`),
so adding a track later only means dropping the file here and adding an entry.

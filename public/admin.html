Default login form (new visitors)

Default entry screen for new visitors, plus (for LAN / dedicated) which server to connect to. The mode applies on a first visit only. LAN server sets the default server fields, never a lock: Automatic uses the page's own address (port 7234, no TLS), Forced uses the host, port and TLS set below. In both cases a player can edit the fields and the server they last connected to is remembered, so only players who never picked one of their own follow a change made here. With Forced, the proxy auto-follows the host and is rebuilt on every reconnect, so the client always reconnects. A forced proxy URL (ws:// or wss://) is pushed to every client and locks the field; hiding the field removes it and its label from the login form. All of this applies to the LAN / dedicated mode only — the Internet (pokerth.net) mode is never affected.
